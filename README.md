## 🛠️ Features
- **Segmented Platform Toggle**: A premium, sliding segmented tab control switcher to toggle between **Android Native** and **Flutter Cross-Platform** views.
- **Dynamic Theming**: Interactive color highlights that shift smoothly between **Android Green** (`#3DDC84`) and **Flutter Cyan** (`#00E5FF`) based on the active view.
- **Real-time Live Search**: Instant interactive filter search across all technical skills, frameworks, and tools.
- **Category Filter Pills**: Fast filtering of skills by core categories (Languages, UI Frameworks, Architecture, DI, Networking, Databases, Testing, Specialized SDKs).
- **Responsive Bento Grid Layout**: Premium, dark-mode design optimized for mobile, tablet, and desktop screens.
- **Interactive Carousel**: Showcasing mobile application features with intuitive sliding cards and pagination.

## 📦 How to Run Locally
Since this is a lightweight, high-performance static website, you can run it directly:
1. Clone the repository:
   ```bash
   git clone https://github.com/ahmed-a-elfattah/ahmed-a-elfattah.github.io.git
   ```
2. Navigate into the directory:
   ```bash
   cd ahmed-a-elfattah.github.io
   ```
3. Open `index.html` in your favorite web browser, or run a simple local server:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.
